# 表现层架构设计

## 一、层定位

表现层（Presentation Layer）是系统与用户之间的直接交互界面，承载所有可视化操作与信息呈现。该层负责将底层智能体、模型与数据的计算结果转化为学生可理解、可操作的界面元素，同时采集用户行为数据回传至数据层，形成闭环。

---

## 二、设计目标

1. **多端统一体验**：Web 端与小程序端在视觉、交互、核心功能上保持高度一致，降低用户切换成本。
2. **低延迟响应**：页面首屏加载 < 1.5s，交互反馈 < 100ms，大模型生成内容采用流式输出。
3. **无障碍与包容性**：支持字体缩放、高对比度模式，适配不同认知风格学生的使用习惯。
4. **可扩展性**：新功能模块可独立发布，支持 A/B 测试与灰度上线。

## 四、核心页面与功能模块

### 4.1 学习工作台（首页）
- **今日任务看板**：基于智能体层推送的个性化学习计划，展示待完成任务、已超时任务、推荐复习内容。
- **智能助手入口**：悬浮快捷按钮，一键唤起多智能体对话界面。
- **学习数据概览**：环形图展示今日学习时长、知识点掌握度、错题分布。

### 4.2 资源中心
- **智能搜索**：支持自然语言搜索（如"帮我找高中导数易错题"），调用智能体层解析意图并返回结构化结果。
- **资源卡片**：展示课程、文档、视频、题库的统一卡片，支持一键加入学习清单。
- **筛选与排序**：按难度、时长、评分、适用画像维度多维筛选。

### 4.3 智能对话界面
- **多智能体切换 Tab**：答疑 Agent、规划 Agent、批改 Agent、陪伴 Agent 快速切换。
- **消息流式渲染**：大模型返回内容逐字显示，支持 Markdown、LaTeX 公式、代码高亮。
- **多模态输入**：支持文字、语音输入、图片上传（拍照搜题、上传笔记）。
- **上下文管理**：显示当前对话关联的学习资料与历史记录，支持手动关联知识点。

### 4.4 个人学习档案
- **学生画像可视化**：雷达图展示六大画像维度得分，支持时间轴对比。
- **知识图谱**：基于数据层知识库生成的个人知识掌握图谱，节点颜色标识掌握程度。
- **错题本**：支持按知识点、错误类型、时间筛选，关联智能体解析与同类题推荐。

### 4.5 社交与激励（轻社交模块）
- **学习小组**：基于画像匹配的学习搭子推荐，支持组队打卡、互相答疑。
- **排行榜与成就**：可选择参与的匿名排行榜、个人成就徽章墙。
- **动态广场**：选择性分享学习笔记、心得，支持点赞与评论。

## 五、与下层交互接口

```
表现层  <--HTTPS/JSON-->  智能体层（RESTful API + WebSocket）
表现层  <--HTTPS/JSON-->  模型层（直接调用多模态生成接口，用于预览与即时渲染）
表现层  <--HTTPS/JSON-->  数据层（用户数据读写、资源库查询）
```

### 关键接口约定
- **统一响应格式**：`{ code: number, data: any, message: string, traceId: string }`
- **鉴权方式**：JWT（Access Token + Refresh Token），小程序额外校验 `openid`。
- **流式响应**：SSE（Server-Sent Events）用于大模型文本生成；WebSocket 用于智能体主动推送与多人协作场景。
- **文件上传**：直传至对象存储（OSS/COS），返回 URL 后提交至业务接口。

## 六、性能与体验优化策略

1. **首屏优化**：SSR（服务端渲染）+ 静态资源 CDN + 图片懒加载与 WebP 格式。
2. **缓存策略**：
   - 本地 IndexedDB 缓存学生画像、知识图谱等低频变更数据；
   - Service Worker 离线缓存核心页面，支持弱网/断网浏览已加载内容。
3. **流式体验**：大模型回答采用打字机效果，首包返回时间 < 500ms，避免用户等待焦虑。
4. **骨架屏与 Loading 状态**：所有异步操作提供明确的进度反馈，避免白屏。
5. **小程序专项**：
   - 分包加载，主包体积 < 2MB；
   - 利用微信 `worker` 线程进行复杂数据预处理；
   - 订阅消息用于学习提醒与计划推送。

---

## 七、安全与隐私

1. **前端敏感操作**：所有涉及学生隐私数据（如真实姓名、学校）的展示需脱敏处理，支持用户一键隐藏。
2. **XSS 防护**：富文本内容（如智能体返回的 Markdown）需经过 DOMPurify 清洗。
3. **小程序合规**：严格遵守《微信小程序平台运营规范》，学习类内容不涉及违规收集个人信息。
