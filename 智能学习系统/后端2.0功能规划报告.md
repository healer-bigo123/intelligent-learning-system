## 五、技术实现要点

### 5.1 大模型 API 切换预留

所有智能生成功能都通过 `LLMClient` 统一调用，切换 API 只需改 `.env`：

```env
# 当前：火山方舟
LLM_PROVIDER=volces
VOLCES_API_KEY=xxx

# 切换：讯飞星火
LLM_PROVIDER=xinghuo
XINGHUO_APP_ID=xxx
XINGHUO_API_KEY=xxx
XINGHUO_API_SECRET=xxx
```

**无需改动业务代码。**

### 5.2 数据库扩展

SQLite 在开发阶段足够，生产环境可无缝迁移到 MySQL/PostgreSQL：

```python
# .env
DATABASE_URL=sqlite:///./data/smart_learning.db  # 开发
# DATABASE_URL=mysql+pymysql://user:pass@localhost/db  # 生产
```

### 5.3 文件存储

| 文件类型 | 存储方式 | 路径 |
|---------|---------|------|
| 上传文档 | 本地磁盘 | `./data/uploads/` |
| 思维导图 | 本地磁盘 + 数据库记录 | `./data/mindmaps/` |
| 生成视频 | 本地磁盘 | `./data/videos/` |
| PPT | 本地磁盘 | `./data/ppts/` |
